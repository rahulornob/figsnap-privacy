Figsnap Privacy Policy

Last updated: April 2026

Overview

Figsnap is a Chrome extension designed to help users capture and export visuals from Figma quickly and efficiently.

Your privacy is important to us. Figsnap is built to be simple, secure, and transparent.

Information We Collect

Figsnap does not collect, store, or share any personal data.

  • No personal information such as name or email is collected
  • No browsing activity is tracked
  • No design files or captured content are stored

All actions occur locally on your device.

How Figsnap Works

Figsnap interacts only with the active Figma tab when you use the extension.

  • Reads content from the current Figma page
  • Allows you to capture or export visuals

This happens only when you actively use the extension.

Permissions

Figsnap requires certain permissions to function properly:

  • activeTab – to access the current Figma tab when you use the extension
  • Host permission (figma.com) – to interact with Figma content
  • scripting – to enable capture functionality within the page

These permissions are used strictly for core functionality.

Third-Party Services

Figsnap does not use any third-party services that collect user data.

Remote Code

Figsnap does not execute any remote code. All functionality runs locally within your browser.

Data Storage

Figsnap does not store user data on servers, databases, or external systems.

Data Sharing

We do not sell, share, or transfer any user data to third parties.

Changes to This Policy

This privacy policy may be updated from time to time. Any changes will be reflected on this page with an updated date.
